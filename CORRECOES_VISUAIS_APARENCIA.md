# Correções visuais e aparência — AgileTask Modelo 5

Alterações realizadas nesta versão:

- Corrigido o fundo branco do Assistente de IA Roby, mantendo a estética escura/glass do Modelo 5.
- Corrigidos os botões de criação de tarefa para terem texto branco e contraste adequado no fundo escuro.
- O botão flutuante “Nova tarefa” também recebeu contraste reforçado.
- A aba Configurações > Aparência agora altera as cores globais do site de forma mais real: gradientes, botões, textos de destaque, bordas, sombras, hero visual, sidebar e progressos passam a respeitar a cor escolhida.
- O layout não foi alterado estruturalmente; apenas foram corrigidas cores, contraste e consistência visual.

Observação: cores de status, como verde para concluído, amarelo para média prioridade e vermelho para atraso, foram mantidas para preservar significado visual.
