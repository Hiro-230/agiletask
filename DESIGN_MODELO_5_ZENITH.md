# AgileTask — Estética Modelo 5 / Zenith

Esta versão aplica uma estética inspirada no Modelo 5 enviado pelo usuário: dashboard web com paisagem/montanhas, glassmorphism, tons azul-ciano, profundidade visual e navegação lateral premium.

## Alterações principais

- Sidebar redesenhada em vidro escuro com gradientes e sensação de profundidade.
- Fundo global com atmosfera de montanha/skyline, remetendo ao painel Zenith.
- Dashboard refeito com hero visual, estatísticas em cards translúcidos, progresso do projeto e prioridades.
- Página de tarefas refeita com cards em glassmorphism, filtros, busca e abertura direta do detalhe.
- Kanban redesenhado com colunas premium, cartões com ações rápidas e visual de sprint.
- Detalhamento da tarefa adaptado ao novo visual, mantendo planejamento por IA.
- Transições e microinterações preservadas para reforçar a ideia de “tarefas ágeis”.

## Build

Validado com:

```bash
npm run build
```

A pasta `dist` já foi gerada para publicação estática na Vercel.
